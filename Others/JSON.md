# Core principles
A JSON object is a structured object that consists on key-value pairs.
# Example template
```JSON
{
	"name": "Bob",
	"age": 14,
	"brothers": {
		"David": {
			"age": 15
		},
		"Martha": {
			"age": 14
		}
	},
	"favourite_numbers": [14, 15, 1, 12],
	"best_friends": ["Luis", "Amily"]
}
```
